# Data-Analysis-P2-

In this project, I’m working with a real-world geospatial dataset of U.S. state boundaries to explore agricultural planning and analysis. While the dataset covers the entire country, I’ve narrowed my focus to four specific states, Tennessee, Kentucky, Alabama, and Georgia, treating them as my primary agricultural fields for this study.
Using a GeoJSON file, I’ll be analyzing the polygon boundaries of these regions to calculate their exact area. From there, I’m assigning specific, realistic yield values to each state ranging from 6.8 to 8.2 tons per hectare to estimate the total potential harvest for each area. It’s a great way to see how geographic data can be transformed into actionable insights for crop management.
